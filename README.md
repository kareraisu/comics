# comics
CLI app for comic loaning using Java and mySQL.
Enter as guest to see available comics or login as admin for full functionality.
Select your language to feel right at home.

Features:
* Awesome intuitive interface (?).
* Comics, People and Loans management (add, delete or edit).
* Robust, fail-safe implementation with plenty of informational messages.
* Scalable design.
* Localization through external text files (see /locales folder).
